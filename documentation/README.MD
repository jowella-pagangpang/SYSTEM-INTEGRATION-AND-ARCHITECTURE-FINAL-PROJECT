# System Integration and Architecture Final Project

## Project Overview

This project integrates the Barangay Health Center Management System and Barangay Information Management System into one centralized platform.

## Integrated Systems

### BHCMS_Web
A web-based Barangay Health Center Management System developed using PHP, HTML, CSS, JavaScript, and MySQL.

### BIMS_Desk
A desktop-based Barangay Information Management System developed using C# Windows Forms and MySQL.

## System Integration Goal

The goal of the project is to synchronize resident information and provide communication between the desktop and web systems using API integration.

## Features

- Resident Information Synchronization
- API Communication
- Dashboard Monitoring
- Health Records Management
- Barangay Resident Management
